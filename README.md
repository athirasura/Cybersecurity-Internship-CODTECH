# Task 2 - Advanced Encryption Tool

## Description
This Python tool encrypts and decrypts files using AES (Fernet) encryption to protect sensitive data.

## Features
- Generate encryption key
- Encrypt files
- Decrypt files

## Tools Used
- Python
- cryptography library (Fernet AES encryption)

## How it Works
1. Generate a key (`key.key`) for encryption/decryption.
2. Encrypt a file by reading its content and converting it into encrypted text.
3. Decrypt the file to restore the original content.

## Outcome
- Successfully encrypts files
- Successfully decrypts files to their original content
