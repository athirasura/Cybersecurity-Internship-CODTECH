# File Integrity Checker

## Description
This project is a Python-based tool that checks file integrity using SHA-256 hashing.

## Features
- Generates hash values for files
- Stores hash values
- Detects file modifications

## How it Works
The program calculates a hash for a file and compares it with a previously stored hash to detect changes.

## Tools Used
- Python
- hashlib library

## Outcome
Successfully detects whether a file has been modified or not.
